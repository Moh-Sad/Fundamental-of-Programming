MY HTML Notes

HTML stands for Hypertext Markup Language.

HTML tags always start with < sign and end with a > tag. These are usually referred to as angle brackets.

markup refers to text that has a special meaning, typically used in the context of HTML (HyperText Markup Language). In HTML, markup is used to instruct the browser on how to display content.

Whitespace: ia a general term for spaces, tabs, and line breaks.

hypertext: is a text that can have links or references.

nesting: is the idea that some bits of code can go inside other bits of code.

syntax: are rules having to do with how the parts of a language can be put together.

void tag (also known as a self-closing tag): is an element that does not require a closing tag. These tags are used to insert elements that do not have any content between an opening and closing tag. Instead, they are self-contained.

mark: is a HTML tag used to highlight words.

sup tag: is used to display text in superscript, which means the text appears slightly above the normal line of text. This is often used for mathematical exponents, footnotes, or other annotations. For e.g. E = MC2.

sub tag: is used to display text in subscript, which means the text appears slightly below the normal line of text. This is commonly used in chemical formulas, mathematical expressions, or to denote specific notations. For e.g. H 2 O.

strong tag: in HTML is used to indicate that the text inside it has strong importance or emphasis. By default, text wrapped in the strong tag is displayed in boldface in most browsers.

Headings: are defined using the h1 to h6 tags, where h1 represents the highest level of heading (usually the main title) and h6 represents the lowest level of heading. Here's a brief overview of each heading tag:

  • h1: Main heading (largest)
  • h2: Subheading (second largest)
  • h3: Sub-subheading
  • h4: Fourth level heading
  • h5: Fifth level heading
  • h6: Sixth level heading (smallest)

Block-level elements: take up the full width available, meaning they start on a new line and stack vertically. Examples: div, p, h1, h2, ul, li, section, and footer.

Inline elements: only take up as much width as necessary and do not start on a new line. They flow within the content of block-level elements. Examples: span, a, strong, em, and img.

div element: stands for "division" and is used as a generic container for other HTML elements. It does not have any semantic meaning on its own but is very useful for structuring and styling web pages.

ul element: is a block-level element that defines an unordered list. Each item in the list is typically marked with a bullet point.

ol element: is a block-level element that defines an ordered list. Each item in the list is typically preceded by a number or letter, indicating its position in the sequence. For Example:

List of Programming Languages

  1. Python
  2. C++
  3. Java
  4. Javascript
  5. C

Elements with Optional Closing Tags:

  • Paragraph
  • Division
  • List of items

Note: When a closing tag is left off in HTML, the browser uses a set of parsing rules to determine when to automatically close the element.

The type attribute for "ol" offers several options for list item markers:

  • "1" for numbers (the default)
  • "A" for uppercase letters
  • "a" for lowercase letters
  • "I" for uppercase Roman numerals
  • "i" for lowercase Roman numerals

Nested lists: are lists that contain other lists as their elements. For Example:

  1. Cars
    • Toyota
    • Nissan
    • BMW
  2. Drinks
    • Coca Cola
    • Pepsi
    • Merinda

URL (Uniform Resource Locator) is used within the "a" tag to specify the destination of the link.

"a" tag in HTML is used to create hyperlinks, which allow users to navigate from one page to another. The "a" tag stands for "anchor," and it is essential for linking web pages together.
